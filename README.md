# App Assets Repository 🖼️

This repository serves as a dedicated host for high-performance images and media assets used across my React applications and other web projects.

## 🚀 Purpose
I created this repository to solve performance issues caused by slow external links. By hosting images directly on GitHub's fast content delivery network (CDN), I ensure:
- **Faster loading times** for the end-user.
- **Reliable uptime** (no broken image links).
- **Better organization** of my project-specific media.

## 🛠️ How to use an image in an App
To link an image from this repository to a project, use the "Raw" URL format:

```html
<img src="https://githubusercontent.com" alt="Trampoline Party" />
```

## 📂 Structure
- `/` (Root): Main images and global assets.
